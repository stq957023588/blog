# 命令

* 查询端口占用命令

```shell
netstat -tunlp|grep 8080
```

* 查看所有进程

```shell
ps aux
```

* 查看磁盘空间

```shell
df -h
```

* 查看文件信息

```shell
stat <文件名称>
```

* 在文件末尾添加字符串

```shell
echo "hello">>test.txt
```

* 查找文件

```shell
find / -name <文件名称>
```

* 查看当前文件夹路径

```shell
pwd
```

* 查看shell解释器类型

```shell
ls -l /bin/sh
```

* 修改解释器类型

```shell
dpkg-reconfigure dash
```

* 删除文件/文件夹

```shell
rm -rf <文件夹名称>   
rm -f <文件名称>
```

* 解压压缩文件

```shell
tar -zxvf 压缩文件名.tar.gz
```
