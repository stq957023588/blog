# Git

## 生成SSH Key

> ssh-keygen -t rsa -C “备注”

## 初始化项目

> git init

## 生成.gitignore文件

> touch .gitignore

## 修改用户信息

> 修改当前项目   
> git config user.name fool   
> git config user.email fool@fool.com  
> 全局修改   
> git --global config user.name fool     
> git --global config user.email fool@fool.com
