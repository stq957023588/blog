# 多层感知机

在Softmax的基础上添加了隐藏层,隐藏层使用激活函数

## 调优

### 默认参数 

| 参数名称   | 值   |
| ---------- | ---- |
| 学习率     | 0.1  |
| 训练轮数   | 10   |
| 隐藏单元数 | 25   |
| 隐藏层数   | 1    |



### 修改学习率(lr)

| 学习率 | 训练耗时           | 预测成功数 |
| ------ | ------------------ | ---------- |
| 0.05   | 370.08942794799805 | 3/6        |
| 0.1    | 371.32980585098267 | 4/6        |
| 0.2    | 339.55971217155457 | 4/6        |
| 0.5    | 298.56382608413696 | 4/6        |

学习率越低,损失下降的快,

### 修改训练轮数(num_epochs)

轮数越多,最终的损失越小
