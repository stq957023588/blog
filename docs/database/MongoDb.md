# MongoDb

一个NoSql数据库

# 备份

```shell
mongodump --uri "" -d [数据库名称] -o [输出位置]
```

```shell
mongodump -h 127.0.0.1:27017 -d [数据库名称] -o [输出位置]
```

# 恢复

```shell
mongorestore -h 127.0.0.1:27017 -d [数据库名称] [备份文件夹位置]
```
