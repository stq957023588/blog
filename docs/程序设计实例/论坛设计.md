# 论坛设计

## 表结构设计

- 用户表

  | 字段         | 类型        | 约束        | 说明                       |
  | ------------ | ----------- | ----------- | -------------------------- |
  | id           | int         | 主键        | 自增ID                     |
  | name         | varchar(32) | 不为空,唯一 | 用户名称                   |
  | email        | varchar32   | 唯一        | 邮箱                       |
  | head_pic     | varchar128  |             | 头像,网络地址              |
  | created_time | timestamp   |             | 创建时间                   |
  | deleted      | int         | 0,1         | 是否删除;0为未删除,1为删除 |
  | delete_time  | timestamp   |             | 逻辑删除时间               |
